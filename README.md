# 🧪 Manual QA Testing Project – SauceDemo

Este repositorio contiene un proyecto de **testing manual** realizado sobre el sitio:

https://www.saucedemo.com/

El objetivo es demostrar habilidades de QA manual incluyendo:

- Test Plan
- Test Cases
- Bug Reports
- Test Execution Report

---

## 🔍 Application Under Test

SauceDemo es una aplicación web de e-commerce creada para practicar testing.

Funciones principales:

- Login de usuario
- Visualización de productos
- Agregar productos al carrito
- Checkout
